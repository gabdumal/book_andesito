Vamos instalar o [**CMake**](https://cmake.org/) por meio dos pacotes distribuídos pelas distribuições tratadas.
Também instalaremos o [**Ninja**](https://ninja-build.org/), um gerador de build com foco em velocidade.

## Pacotes

### Fedora e WSL

Para instalar o **CMake** no Fedora ou no WSL usando o Fedora, **execute** o seguinte comando no terminal.

```zsh title="Fedora ou WSL" frame="terminal"
sudo dnf upgrade -y
sudo dnf install cmake ninja-build python3-ninja -y
cmake --version
# cmake version ?.??.?
ninja --version
# 1.12.1
```

### Mint

Para instalar o **CMake** no Mint ou em outras distribuições baseadas no Debian, **execute** o seguinte comando no terminal.

```zsh title="Mint" frame="terminal"
sudo apt update -y && sudo apt upgrade -y
sudo apt install cmake ninja-build -y
cmake --version
# cmake version 3.28.3
ninja --version
# 1.11.1
```

## Extensões

Para trabalhar com o **CMake** no Visual Studio Code, é recomendado instalar algumas extensões que são inclusas no pacote C/C++ Extension Pack.

Selecione o **perfil** `C/C++` no VSCode, que criamos anteriormente no subcapítulo sobre [C](../../compiler/vscode).

Pesquise pela extensão [C/C++ Extension Pack](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools-extension-pack) e a **instale**.

Esse pacote inclui as seguintes extensões:

- [C/C++](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools): fornece suporte para desenvolvimento em C e C++.
- [C/C++](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools-themes) Themes: fornece temas de cores para o Visual Studio Code.
- [CMake Tools](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cmake-tools): fornece suporte para desenvolvimento e ferramentas para trabalhar com projetos CMake.
