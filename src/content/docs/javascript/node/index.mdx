O [**Node**](https://nodejs.org/) é um ambiente de execução para JavaScript que permite rodar códigos JavaScript fora do navegador.

Utilizando o PNPM, é possível instalar versões específicas do Node, e trocar entre elas para diferentes projetos.

## Instalação

O processo de instalação é o mesmo no Fedora (e no WSL usando o Fedora) e no Mint.

**Abra** o terminal e **execute** o seguinte comando.
Ele instalará a versão mais recente do Node que é considerada estável.

```zsh title="Fedora, WSL ou Mint" frame="terminal"
pnpm env use --global lts
```

O resultado será como a seguir.

```zsh title="Fedora, WSL ou Mint" frame="terminal"
# Fetching Node.js 24.14.1 ...
# Node.js 24.14.1 was installed
#   /home/gabriel/.local/share/pnpm/nodejs/24.14.1
# Node.js 24.14.1 was activated
# /home/gabriel/.local/share/pnpm/node -> /home/gabriel/.local/share/pnpm/nodejs/24.14.1/bin/node
```

**Feche** e **reabra** o terminal, e execute o comando abaixo para testar se a instalação foi bem sucedida.

```zsh title="Fedora, WSL ou Mint" frame="terminal"
node --version
# v24.14.1
```
