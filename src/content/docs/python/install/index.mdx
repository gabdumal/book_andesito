O [**Python**](https://www.python.org/) é instalado por padrão no Linux, bastando verificar se as ferramentas auxiliares estão instaladas.

Para isso, execute o seguinte comando no Fedora.

```zsh title="Fedora ou WSL" frame="terminal"
sudo dnf install python3 -y
python3 --version
# Python 3.13.3
```

Ou este no Mint e derivados do Debian.

```zsh title="Mint" frame="terminal"
sudo apt install python3 -y
python3 --version
# Python 3.13.3
```

## Pip

O [**Pip**](https://pip.pypa.io/en/stable/installation/) é um gerenciador de pacotes para o Python.

Em geral, ele já vem instalado junto com o Python, mas você pode garantir sua atualização executando o seguinte comando.

```zsh title="Fedora, WSL ou Mint" frame="terminal"
python3 -m pip install --upgrade pip
# Successfully installed pip-25.0.1
```
