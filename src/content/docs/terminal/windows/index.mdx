Este subcapítulo aborda o [**Windows Terminal**](https://github.com/microsoft/terminal), um emulador de terminal moderno para usuários do Windows.
Acesse sua página na [Microsoft Store](https://www.microsoft.com/store/productId/9N0DX20HK701?ocid=pdpshare) e instale-o.

## Atalho de teclado

Vamos criar um atalho de teclas para abrir o Windows Terminal.
Abra o `Executar` com <Kbd linux={"Super+R"} windows={"Win+R"} />, digite `shell:AppsFolder`, e pressione <Kbd linux={"Enter"} windows={"Enter"} />.

<Figure
  caption="Abrindo a pasta de aplicativos pelo Executar."
  images={[
    {
      src: openFolder,
      alt: "Janela do programa Executar, em que se digita o texto supracitado.",
    },
  ]}
/>

Encontre o ícone do Windows Terminal, clique com o botão direito do mouse e selecione `Criar atalho`.

<Figure
  caption="Criando um atalho para o Windows Terminal."
  images={[
    {
      src: selectTerminal,
      alt: "Janela do Explorador de Arquivos aberto da pasta de aplicações do sistema. A entrada do Windows Terminal foi selecionada, e clicada com o botão direito do mouse. Aparece ao seu lado um menu de opções.",
    },
  ]}
/>

Clique em `Sim` para criar o atalho na área de trabalho.

<Figure
  caption="Confirmação da criação do atalho na área de trabalho."
  images={[
    {
      src: confirm,
      alt: "Janela de confirmação da criação do atalho na área de trabalho.",
    },
  ]}
/>

Agora, clique com o botão direito do mouse no atalho criado na área de trabalho e selecione `Propriedades`.

<Figure
  caption="Abrindo as propriedades do atalho do Windows Terminal."
  images={[
    {
      src: editShortcut,
      alt: "Menu de opções do atalho do Windows Terminal.",
    },
  ]}
/>

Clique sobre a caixa de texto `Tecla de atalho` e pressione a combinação de teclas <Kbd linux={"Ctrl+Alt+T"} windows={"Ctrl+Alt+T"} />.

<Figure
  caption="Definindo a combinação de teclas para abrir o Windows Terminal."
  images={[
    {
      src: setKeybinding,
      alt: `Janela de propriedades do atalho do Windows Terminal, com a caixa de texto "Tecla de atalho" selecionada. Dentro dela foram pressionadas as teclas "Ctrl + Alt + T".`,
    },
  ]}
/>

Clique em `Aplicar` e depois em `OK`.
Agora, sempre que você pressionar <Kbd linux={"Ctrl+Alt+T"} windows={"Ctrl+Alt+T"} />, o Windows Terminal será aberto.

## Perfis

O Windows Terminal permite a criação de perfis, que são configurações personalizadas para o terminal.

Para acessar as configurações, clique no ícone de seta para baixo no canto superior direito da janela e selecione `Configurações`.

<Figure
  caption="Menu de abertura das configurações do Windows Terminal."
  images={[
    {
      src: openConfigurations,
      alt: "Janela do Windows Terminal, em que foi selecionado o botão de abrir o menu de contexto para abrir um novo perfil ou as configurações.",
    },
  ]}
/>

As configurações são divididas em seções, que permitem configurar o tema, as fontes, os atalhos de teclado, os perfis e outros aspectos do terminal.

<Figure
  caption="Tela inicial das configurações do Windows Terminal."
  images={[
    {
      src: initialConfigs,
      alt: `Janela de configurações do Windows Terminal, em que está aberta a primeira tela, referente a "Inicialização".`,
    },
  ]}
/>

Eu gosto de usar o tema claro no terminal, o que eu posso fazer pela seção `Aparência`.
Então, posso selecionar a definição `Tema do Aplicativo` como `Claro`.

<Figure
  caption="Definição do tema claro no Windows Terminal."
  images={[
    {
      src: lightTheme,
      alt: "Janela de configurações do Windows Terminal, em que está aberta a tela de aparência, com a opção de tema claro selecionada.",
    },
  ]}
/>

Entretanto, essa definição apenas muda o tema do aplicativo.
Para configurar a cor de fundo, as cores do texto, fonte, e aspectos relacionados, é necessário editar um perfil.

Para aplicar a mesma configuração a todos os perfis, selecione a seção `Padrões` dentro de `Perfis` no menu lateral.
Então selecione a opção `Aparência` no fim do menu.

Eu defini o tema como `One Half Light` e o tamanho da fonte como 16.

<Figure
  caption="Definições do perfil padrão."
  images={[
    {
      src: defaultProfile,
      alt: `Janela de configurações do Windows Terminal, em que está aberta a tela dos padrões do terminal, na seção de "Aparência".`,
    },
  ]}
/>

Toda essa interface apenas edita um arquivo de configuração, que pode ser acessado diretamente pelo botão `Abrir o arquivo JSON`, que fica no canto inferior esquerdo da janela.

Como já instalamos o Visual Studio Code, ele será usado para abrir o aqrquivo de configurações.

<Figure
  caption="Arquivo JSON de definições do Windows Terminal."
  images={[
    {
      src: jsonFile,
      alt: "Trecho do arquivo JSON de definições do Windows Terminal aberto no Visual Studio Code.",

Eu gosto de usar um tema bastante popular chamado [**Catppuccin**](https://catppuccin.com/).
Ele é um projeto que oferece personalização para uma série de programas.
Você pode optar entre quatro variantes: Latte, Frappé, Macchiato e Mocha.

Para aplicar o tema ao Windows Terminal, acesse o [repositório da implementação](https://github.com/catppuccin/windows-terminal).
Decida qual das variantes você deseja usar.

Eu prefiro a variante `Latte`.
Para aplicar o tema, abra o arquivo `latte.json` do repositório do GitHub (ou o correspondente à variante que você escolheu) e copie o conteúdo.

Então, abra o arquivo de configuração do Windows Terminal e cole o conteúdo copiado dentro da seção `schemes`.

<Figure

/>

Depois, copie o conteúdo do arquivo `latteTheme.json` do repositório (ou o correspondente à variante que você escolheu) e cole dentro da seção `themes`.

<Figure
  caption={`Tema Catppuccin Latte sendo aplicado no atributo "themes".`}

/>

Salve o arquivo de configuração e feche o Windows Terminal.
Abra o terminal novamente.
Agora, você verá que o esquema de cores `Catppuccin Latte` está disponível na lista de temas.

Você pode aplicar o tema da mesma forma como selecionamos o tema claro e o `One Half Light` anteriormente.

## Configurando o WSL

Tendo instalado o WSL no computador, você verá uma nova entrada para o Fedora na aba de perfis do **Windows Terminal**.

<Figure
  caption="Opção do Fedora na lista de perfis do Windows Terminal."
  images={[
    {
      src: profiles,
      alt: `Lista de perfis do Windows Terminal, em que são exibidas as opções: "Windows PowerShell", "Prompt de Comando", "Azure Cloud Shell" e "FedoraLinux-42".`,
    },
  ]}
/>

Eu acho mais prático ter o Fedora como padrão quando inicio o terminal.
Para isso, abra as configurações do Windows Terminal.
Na seção `Startup` (ou `Inicialização`), você pode definir o perfil padrão.
Selecione a entrada do Fedora, que tem o ícone azul com uma letra "F" estilizada, e clique em `Salvar`.

<Figure
  caption="Definindo o perfil do Fedora como a opção de inicialização padrão do Windows Terminal."
  images={[
    {
      src: fedoraAsDefault,
      alt: "Configurações do Windows Terminal, em que se define o perfil padrão de inicialização.",
    },
  ]}
/>

Outra coisa que eu gosto de fazer é ocultar os perfis que não uso.
Pra isso, acesse o perfil que deseja ocultar, e ative a opção `Ocultar perfil da lista suspensa`.
Podemos fazer isso para o `Prompt de comando` e para o `Azure Cloud Shell`.

Não se esqueça de **salvar** as definições depois de editá-las.
